# Gaming Hub

A collection of mini-games built with C++ and SFML as part of an OOP course assignment at FAST University.

## Games Included
- **Tic Tac Toe** — Classic two-player strategy game
- **Pong** — Two-player arcade paddle game
- **Memory Match Cards** — Card flipping memory game

## Features
- Graphical main menu with game selection
- Player vs Player multiplayer mode
- Centralized leaderboard system
- Persistent score storage using file handling
- Leaderboard updates dynamically after each match

## OOP Concepts Applied
- Encapsulation
- Modular class design (separate class per game)
- Game state management
- File handling via dedicated LeaderboardManager class

## Technologies Used
- C++
- SFML (Simple and Fast Multimedia Library)

## How to Run
1. Install SFML on your system
2. Clone the repository
3. Compile and link with SFML
4. Run the executable to launch the hub
